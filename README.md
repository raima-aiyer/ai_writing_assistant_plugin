# AI Writing Assistant Chrome Extension

A Chrome extension that uses Google's Gemini Flash 2.0 AI to help improve your writing across any website. The extension provides writing improvements, grammar checks, and text rewriting in various tones.

## Features

- Writing improvement suggestions
- Grammar checking and corrections
- Text rewriting in different tones:
  - Formal
  - Casual
  - Persuasive
  - Friendly
  - Professional
- Works in any text input field or textarea
- Supports Gmail, LinkedIn, and other websites

## Installation

1. Clone or download this repository
2. Open Chrome and go to `chrome://extensions/`
3. Enable "Developer mode" in the top right corner
4. Click "Load unpacked" and select the extension directory
5. The extension icon should appear in your Chrome toolbar

## Setup

1. Click the extension icon in your Chrome toolbar
2. Enter your Google Gemini API key in the popup window
3. The API key will be saved automatically for future use

## Usage

1. Select text on any webpage or click into a text input field
2. Click the extension icon
3. Choose your desired tone
4. Click one of the following buttons:
   - "Improve Writing" for general improvements
   - "Check Grammar" for grammar corrections
   - "Rewrite" to rewrite the text in your chosen tone

## Requirements

- Google Chrome browser
- Google Gemini API key (get one from Google AI Studio)

## Security

- Your API key is stored securely in Chrome's storage
- The extension only sends selected text to the Gemini API
- No data is stored or shared with third parties
